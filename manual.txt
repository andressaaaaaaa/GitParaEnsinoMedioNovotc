primeiro, criamos no github um repositório como nome GitParaEnsinoMedioNovotc.

-seguinte, fizemos um git clone do repositório para nosso PC

-criamos um arquivo um arquivo chamado manual.txt

-configuramos o nome do usuario e o e-mail do usuario

-fizemos o commit e finalmente um push na Master

mostramos os códigos seguidos:
git clone (url do repositório)
git config --global user.name "Tiffany"(uma vez só no git)
git config --global user.email "tiffanyandressa15@gmail.com"(uma vez só no git)
git status
git add .
git commit -m "digitamos um texto que npossa descrever o que a gente fez"
git status
git push (precisaremos depois deste passo para passar o nosso usuário e senha do github) 

segundo passo no git 
criação de branchs(galhos de uma árvore)


O uso de branchs é para criar cópias do projetp que se encintram na master (ou cópias de outras branchs )


A continuação segue os comandos para criar uma branch no PC e no github

git status (para conferir se tem arquivos para commitar)
(se tiver arquivos não adicionados, repetir o passo número 1)

git checkout -b "no da branch" 
(fazendo isso a gente muda automaticamente da master para a nova branch)

Fazendo todos os comandos anteriores estaremos em uma cópia da master
Agora podemos editar os arquivos desta cópia e depois fazemos o passo número 1
Cuidado quando vamos fazer o push devemo saber, que no Github não existe essa branch
Então fazendo o git push, aparecerá uma recomendação, que devemos implentar

git push --set-upstream origin "nome da branch"
